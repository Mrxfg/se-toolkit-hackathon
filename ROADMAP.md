# 🗺️ CarBot Roadmap

## ✅ V1 - MVP (Current Version)
- Basic car listing (add/edit/delete)
- Search by brand/model
- Browse by brand
- Location support (GPS/city)
- Pagination
- User authentication
- Ownership verification

## 🚀 V2 - Real Product (Planned)

### 📸 Image Upload
- Upload multiple photos per car listing
- Image storage and retrieval
- Photo gallery in car details
- Image compression and optimization

### ❤️ Favorites
- Save favorite car listings
- View saved favorites
- Remove from favorites
- Favorite count per listing

### 📞 Contact Seller
- Direct messaging between buyer and seller
- Phone number sharing (optional)
- In-app chat system
- Contact request notifications

### 🔔 Notifications
- New listings matching user preferences
- Price drops on favorited cars
- Messages from potential buyers
- Listing expiration reminders

### 🎯 Better Search
- Advanced filters (price range, year range, mileage)
- Sort by price, date, mileage
- Location-based search (nearby cars)
- Save search preferences
- Search history

## 🔮 Future Ideas
- User ratings and reviews
- Verified sellers badge
- Car comparison feature
- Price history tracking
- Integration with car valuation APIs
- Multi-language support
